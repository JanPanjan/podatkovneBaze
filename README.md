## Ocenjevanje

| tip | vrednost |
| -------------- | --------------- |
| izpit / kolokvij | 60% |
| domače naloge | 40% |
| ustni izpit | 20% |

Izpit ali 2 kolokvija.

### Domače naloge

- 3 naloge
- Rok je 10 dni
- vsak dan zamude odšteje 10% ocene
- Povprečje mora bit >50%

### Pisni izpit

- 4 naloge iz vaj in predavanj
- ura in pol
- več kot 50%

### Ustni izpit

- po opravljenem pisnem izpitu in domačih nalogah
- 3 vprašanja iz predavanj in vaj
- več kot 50%
